# 🌟 星空日记 (Star Diary)

> 每篇日记化作一颗星星，挂在夜空中。写下回忆，点亮属于你的星空。

---

## 📱 手机版（在线）

**👉 [点击打开](https://wilsonss-huang.github.io/star-diary/)** — 手机/电脑浏览器都能用

> 打开后，手机浏览器菜单 →「添加到主屏幕」→ 就像 App 一样！

## 💻 Windows 桌面版（下载）

**👉 [下载最新版](https://github.com/wilsonss-huang/star-diary/releases/latest)** — 下载 `StarDiary-v1.0.0.zip`

> 解压后双击 `星空日记.exe` 即可运行，无需安装！

---

## ✨ 功能

- 📝 写日记 → 一颗星星出现在夜空中
- 😊 6 种情绪颜色（开心/难过/兴奋/平静/心动/沉思）
- 🔍 搜索日记，匹配的星星会高亮连线
- 🌌 3D 星空，鼠标拖拽旋转缩放
- 📱 支持 PWA，手机可添加到主屏幕

## 🛠 技术栈

React 19 · Three.js · TypeScript · Tailwind CSS · Framer Motion · Electron
